## [4] 症状チェック

### 見出し
```
こんな社員、いませんか？
```

### チェック項目

**□ 火が消えたベテラン**
```
かつてはエースだったのに、今は惰性で働いている。
「俺はもういいよ」「若い奴に任せる」が口癖。
技術は持っているのに、若手に教えようとしない。
```

**□ 変化を拒むシニア**
```
「昔からこうやってきた」と自分のやり方に固執。
新しいシステムや手法を受け入れない。
会議では否定から入り、若手を萎縮させている。
```

**□ 燻っている中堅**
```
能力はあるのに、出し惜しみしている。
「どうせ言っても無駄」「頑張っても評価されない」と諦め顔。
入社した頃の情熱が、どこかに消えてしまった。
```

## [5] 2カラム

### 左カラム：リスク訴求

#### 見出し
```
放置すると、どうなるか
```

#### 本文
```
・ベテランの「やる気のなさ」が、若手に伝染する
  →「ああはなりたくない」と、優秀な若手が辞めていく

・技術が継承されないまま、定年を迎える
  →30年のキャリアが、会社に何も残らない

・燻った中堅が、そのまま管理職になる
  →火のないリーダーの下で、チームも燃えない
```

### 右カラム：理論背景

#### 見出し
```
なぜ「没入パターン」なのか
```

#### 本文
```
「やる気を出せ」と言っても、人は動きません。

でも、「自分が没入できる条件」を知れば、
スイッチの入れ方が分かります。

この研修は——
・「消えた火」の原因を特定し
・かつての「没入パターン」を思い出させ
・今の業務で再現する道筋をつけます

「やる気がない」のではなく、
「やる気の出し方を忘れた」だけ。
そう考えると、打ち手が見えてきます。
```

  ——研修後も継続して活用できる運用ツール
```

## [9] 2カラム

### 左カラム：irodoriの強み

irodori研修の特徴
```

#### 本文
```
① 没入起点

```

#### 本文
```

  「やる気を出せ」は逆効果。
  反発か、表面的な服従しか生まない。

【配置転換】
  環境を変えても、本人が変わらなければ
  同じことの繰り返し。

【一般的な研修】
  「新しい知識」を入れても、
  ベテランは「知ってる」で終わる。

```
「そういえば、30代の頃は
 新しい加工方法を誰よりも早く試してたな...」

「あの頃は"誰もやってないこと"をやるのが楽しかった」

「今、"誰もやってないこと"って何だろう...」
```

#### After
```
「若手に"俺の失敗談"を話すようになった。
 成功より失敗の方が学びがあるから」

「新しい設備の導入プロジェクト、俺がリーダーやるよ。
 昔取った杵柄ってやつだ」

「定年までの6年、"守り"じゃなくて"仕上げ"にしたい」
```

### 注釈
```
※上記は研修で期待される変化のイメージです。
 実際の効果は個人・組織によって異なります。
