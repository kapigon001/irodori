## 使い方

### 1. プレビュー
```bash
# ブラウザで開く
open flyer-complete.html
```
または、ファイルをダブルクリックしてブラウザで開いてください。

### 2. PDF保存（印刷用）
1. ブラウザで `flyer-complete.html` を開く
2. `Cmd + P`（Mac）または `Ctrl + P`（Windows）で印刷ダイアログを開く
3. 「送信先」で「PDFに保存」を選択
4. 「その他の設定」で以下を確認：
   - 用紙サイズ: A4
   - 余白: なし
   - 背景のグラフィック: オン
5. 保存

### 3. テキスト編集
HTMLファイルを任意のテキストエディタで開いて編集できます。

**よく編集する箇所:**
- 連絡先: `03-XXXX-XXXX` → 実際の電話番号
- メール: `contact@irodori.com` → 実際のメールアドレス
- URL: `irodori.com/contact` → 実際のURL
- 代表者名: `[代表者名]` → 実際の名前
- 住所: `〒XXX-XXXX 東京都〇〇区〇〇` → 実際の住所

### 4. 配色変更
HTMLファイル冒頭の `:root` セクションで色を変更できます：

```css
:root {
  --color-main: #1a365d;      /* メインカラー（ネイビー） */
  --color-accent: #dd6b20;    /* アクセントカラー（オレンジ） */
  /* ... */
}
```

## デザイン仕様

| 項目 | 仕様 |
|------|------|
| サイズ | A4（210mm × 297mm） |
| カラー | フルカラー（CMYK変換は印刷時に対応） |
| フォント | ヒラギノ角ゴ / 游ゴシック / メイリオ |
| 配色 | ネイビー (#1a365d) + オレンジ (#dd6b20) |

---

## 印刷発注時の注意

1. **解像度**: ブラウザからのPDF出力は画面解像度のため、商業印刷には向かない場合があります
2. **塗り足し**: 必要な場合は印刷会社に相談してください
3. **高品質印刷が必要な場合**: このHTMLを参考に、Illustrator/InDesignで再作成することを推奨
