# Omoro Theory ロジックデザイン分析レポート

**作成目的**: Omoro Theory（オモロ理論）と既存B2B研修理論の体系的比較分析

## 1. B2B研修理論の全体像（ロジックツリー）

### 1.1 理論体系の5系譜

```
B2B研修理論
├── 1. 学習理論系
│   ├── 成人学習理論（Knowles）
│   │   ├── 前提：成人は自己主導的に学ぶ
│   │   ├── 方法：経験を学習資源として活用
│   │   └── 目標：実生活への即時適用
│   │
│   ├── 経験学習サイクル（Kolb）
│   │   ├── Step1：具体的経験（Concrete Experience）
│   │   ├── Step2：内省的観察（Reflective Observation）
│   │   ├── Step3：抽象的概念化（Abstract Conceptualization）
│   │   └── Step4：能動的実験（Active Experimentation）
│   │
│   ├── 変容学習（Mezirow）
│   │   ├── 核心：既存の前提を批判的に省察
│   │   ├── 方法：ディスオリエンティング・ジレンマの経験
│   │   └── 目標：世界観のパラダイムシフト
│   │
│   └── 社会的学習理論（Bandura）
│       ├── 核心：他者の観察を通じた学習
│       ├── 要素：注意・保持・再生・動機づけ
│       └── 目標：自己効力感の向上
│
├── 2. 動機づけ理論系
│   ├── 自己決定理論（Deci & Ryan）
│   │   ├── 三欲求：自律性（Autonomy）
│   │   ├── 三欲求：有能感（Competence）
│   │   ├── 三欲求：関係性（Relatedness）
│   │   └── 目標：内発的動機の最大化
│   │
│   ├── フロー理論（Csikszentmihalyi）
│   │   ├── 条件1：明確な目標
│   │   ├── 条件2：即座のフィードバック
│   │   ├── 条件3：挑戦とスキルのバランス
│   │   └── 状態：没入・時間感覚の消失・自己意識の低下
│   │
│   └── 内発的動機付け理論
│       ├── 起点：興味・関心・好奇心
│       ├── 対比：外発的動機（報酬・罰則）
│       └── 特徴：持続性が高い
│
├── 3. 組織開発系
│   ├── 学習する組織（Senge）
│   │   ├── システム思考（Fifth Discipline）
│   │   ├── 自己マスタリー
│   │   ├── メンタルモデル
│   │   ├── 共有ビジョン
│   │   └── チーム学習
│   │
│   ├── アクションラーニング（Revans）
│   │   ├── 公式：L = P + Q（学習 = 知識 + 質問）
│   │   ├── 方法：実際の問題を題材に学習
│   │   └── 特徴：振り返りと実践の循環
│   │
│   └── 組織開発（OD）
│       ├── 介入：計画的変革プロセス
│       ├── 対象：組織文化・構造・プロセス
│       └── 手法：サーベイフィードバック、チームビルディング
│
├── 4. 能力開発モデル系
│   ├── 70:20:10モデル（Lombardo & Eichinger）
│   │   ├── 70%：経験からの学習（OJT）
│   │   ├── 20%：他者からの学習（メンタリング、コーチング）
│   │   └── 10%：正式な学習（研修、読書）
│   │
│   ├── コンピテンシーモデル
│   │   ├── 定義：高業績者の行動特性
│   │   ├── 用途：採用・評価・育成の基準
│   │   └── 問題点：過去の成功パターンの固定化
│   │
│   └── ストレングスベースアプローチ（Gallup等）
│       ├── 前提：弱みを補うより強みを伸ばす
│       ├── 方法：強み診断（StrengthsFinder等）
│       └── 目標：強みを活かした役割設計
│
└── 5. ポジティブ心理学系
    ├── PERMA理論（Seligman）
    │   ├── P：Positive Emotion（ポジティブ感情）
    │   ├── E：Engagement（没頭）
    │   ├── R：Relationships（関係性）
    │   ├── M：Meaning（意味）
    │   └── A：Achievement（達成）
    │
    └── エンゲージメント理論（Kahn, Schaufeli等）
        ├── 要素：活力（Vigor）
        ├── 要素：熱意（Dedication）
        ├── 要素：没頭（Absorption）
        └── 対比：バーンアウト（燃え尽き）
```

## 2. Gap Analysis：Omoro Theory vs 既存理論

### 2.1 共通点マトリクス

| 既存理論 | 共通する概念 | Omoro Theoryでの位置づけ | 活用の仕方 |
|----------|-------------|-------------------------|-----------|
| **フロー理論** | 没入状態 | 「おもろい」の学術的根拠 | 偶発的フローを「意図的に再現可能な没入パターン」に変換 |
| **自己決定理論** | 自律性・有能感・関係性 | Will/Can/Must統合の三欲求対応 | 自律性→Will、有能感→Can、関係性→Must |
| **経験学習サイクル** | 体験起点の学習 | 「体験起点の気づき」の方法論基盤 | ゲーム体験→内省→パターン抽出→業務転用 |
| **ストレングスベース** | 強みを活かす発想 | 欠損モデルの否定 | 「強み」を「没入パターン」に置換・具体化 |
| **PERMA理論** | Engagement（没頭） | 「おもろい」の心理学的根拠 | 没頭を業務成果に接続する設計を追加 |
| **学習する組織** | 個人と組織の相互作用 | 「改革の両輪」の組織側 | 個人改革と組織改革の同時進行を明示化 |

### 2.2 決定的差異（USP）マトリクス

| 比較軸 | 既存理論の立場 | Omoro Theoryの立場 | 差異の本質 | 実務上の意味 |
|--------|---------------|-------------------|-----------|-------------|
| **起点** | スキル/知識の不足（欠損モデル） | 「おもろい」という感情（充足モデル） | **欠損→充足へのパラダイムシフト** | 受講者のモチベーションが根本的に変わる |
| **変革方向** | 組織→個人（トップダウン適応） | 個人⇔組織（双方向接続） | **適応→接続（Alignment）** | 個人の主体性を維持しながら組織成果を実現 |
| **変革対象** | 個人 OR 組織（どちらか一方） | 個人 AND 組織（両輪同時） | **部分最適→全体最適** | 片方だけ変えても効果が出ない問題を解決 |
| **目標設定** | Must（やるべきこと）の遂行 | Will/Can/Mustの統合 | **義務→統合** | やらされ感を排除し、持続的行動を実現 |
| **持続性** | 研修時のみ効果（一過性） | 自己再現可能（持続性） | **一過性→持続性** | 研修効果の減衰問題を解決 |
| **対象の定義** | 「スキル」「コンピテンシー」 | 「没入パターン」 | **抽象→具体** | 観察可能・再現可能な行動に落とし込む |
| **フローの扱い** | 偶発的に生じる状態 | 意図的に設計・再現する状態 | **偶発→設計** | フローを待つのではなく作り出す |

### 2.3 理論別 詳細Gap Analysis

#### 2.3.1 フロー理論との差異

| 観点 | フロー理論（Csikszentmihalyi） | Omoro Theory |
|------|------------------------------|--------------|
| フローの発生 | 偶発的・状況依存的 | 意図的・設計可能 |
| 対象 | 個人の心理状態の研究 | 個人と組織の変革実践 |
| 業務との接続 | 言及なし（学術研究） | 明示的に業務転用を設計 |
| 組織への展開 | なし | 「組織改革」として明示 |
| 成果物 | フロー状態の理解 | 没入パターンシート、アクションプラン |

**差異の本質**: フロー理論が「フローとは何か」を解明したのに対し、Omoro Theoryは「フローをどう作り出し、業務に活かすか」を設計する。

#### 2.3.2 自己決定理論との差異

| 観点 | 自己決定理論（Deci & Ryan） | Omoro Theory |
|------|----------------------------|--------------|
| 三要素の関係 | 並列（三欲求を満たす） | 統合（重なりを作る） |
| 組織要求 | 外発的動機として扱う | Must（求められる）として統合対象に含める |
| 具体的方法論 | 提示なし（理論モデル） | ワーク・ゲームで具体化 |
| ゴール | 内発的動機の充足 | Will/Can/Mustが重なる状態の設計 |

**差異の本質**: 自己決定理論が「なぜ人は動くか」を説明したのに対し、Omoro Theoryは「動きたい・動ける・求められるを重ねる」という設計論を提示する。

#### 2.3.3 ストレングスベースアプローチとの差異

| 観点 | ストレングスベースアプローチ | Omoro Theory |
|------|---------------------------|--------------|
| 発見対象 | 強み（Strength） | 没入パターン |
| 発見方法 | 診断ツール（StrengthsFinder等） | ゲーム体験→内省→抽出 |
| 定義の具体性 | 34資質等のカテゴリ | 個人固有の行動パターン |
| 業務接続 | 強みを活かせる役割へ配置 | 没入パターンを業務アクションに落とし込む |
| 組織側の変革 | 限定的 | 両輪として明示 |

**差異の本質**: ストレングスベースが「強みを見つけて活かす」に留まるのに対し、Omoro Theoryは「没入パターンを型化し、業務に転用し、組織環境も変える」という三段階を設計する。

## 3. 「おもろい（感情）→ 成果（論理）」アプローチの特異性

### 3.1 アプローチの構造図

```
【従来のアプローチ】
Must（組織要求）→ スキル習得 → 行動変容 → 成果
    ↓
問題：やらされ感 → モチベーション低下 → 効果減衰

【Omoro Theoryのアプローチ】
おもろい（感情）→ 没入パターン（行動）→ 業務転用（設計）→ 成果（論理）
    ↑                                                      ↓
    └──────────── 正のフィードバックループ ─────────────────┘
```

### 3.2 変換メカニズムの詳細

#### Stage 1: 感情の捕捉（おもろい → 認識）

| 要素 | 説明 |
|------|------|
| **入力** | 「おもろい」という主観的感情体験 |
| **方法** | ゲーム・ワークで意図的に没入状態を生成 |
| **出力** | 「自分は今、没入していた」という自己認識 |
| **従来との違い** | 従来は感情を「測定困難」として無視。Omoro Theoryは感情を起点として活用 |

#### Stage 2: 行動の型化（没入状態 → 没入パターン）

| 要素 | 説明 |
|------|------|
| **入力** | 没入状態の体験とその認識 |
| **方法** | 振り返りワーク：「何が、どうあったから没入できたか」を言語化 |
| **出力** | 没入パターンシート（観察可能な行動と条件の記述） |
| **従来との違い** | フロー理論は没入を「状態」として研究。Omoro Theoryは「再現可能なパターン」に変換 |

#### Stage 3: 業務への接続（没入パターン → 業務アクション）

| 要素 | 説明 |
|------|------|
| **入力** | 言語化された没入パターン |
| **方法** | マッチングワーク：没入パターン × 現業務の交差点を探索 |
| **出力** | アクションプラン（具体的な業務行動の変更案） |
| **従来との違い** | 従来は「スキルを業務に適用」。Omoro Theoryは「没入パターンを業務に転用」 |

#### Stage 4: 成果の創出と循環（業務アクション → 成果 → おもろい）

| 要素 | 説明 |
|------|------|
| **入力** | 没入パターンを活かした業務行動 |
| **方法** | 実務での実践、成果の測定 |
| **出力** | 業務成果 + 「おもろかった」という感情体験 |
| **従来との違い** | 従来は成果で終了。Omoro Theoryは成果が次の「おもろい」を生む正のループを設計 |

### 3.3 特異性の言語化

#### 特異性1: 感情を「測定困難な主観」から「設計可能な起点」に変換

> **従来の研修設計**: 「成果を出すにはスキルが必要」→「スキルを教える」→「行動が変わるはず」
>
> **Omoro Theoryの研修設計**: 「おもろいと感じると没入する」→「没入パターンを発見する」→「業務で再現する」→「成果が出る」

**なぜこれが特異なのか**:
- 従来の研修は「スキル不足」を前提とし、受講者を「不足を補われる側」に位置づける
- Omoro Theoryは「おもろさへの感受性」を前提とし、受講者を「自分の没入を発見する側」に位置づける
- この違いが、受講者のモチベーションと主体性を根本的に変える

#### 特異性2: 個人の感情と組織の成果を「二項対立」から「同時実現」に再定義

> **従来の二項対立**: 「楽しさ」vs「成果」（楽しいだけでは成果が出ない、成果を出すには楽しさを犠牲にする）
>
> **Omoro Theoryの統合**: 「おもろい × 成果」（楽しいから没入する、没入するから成果が出る、成果が出るからもっと楽しい）

**なぜこれが特異なのか**:
- 従来の研修は「成果のために我慢する」という暗黙の前提があった
- Omoro Theoryは「おもろいから成果が出る」という因果関係を明示し、設計する
- 「やりがい」と「成果」のトレードオフを解消する

#### 特異性3: 研修の効果を「一過性」から「自己再現可能」に変換

> **従来の効果持続**: 研修直後は意識が高まる → 1週間で元に戻る → 効果減衰
>
> **Omoro Theoryの効果持続**: 没入パターンを自分で言語化 → 業務で自分で再現できる → 持続・拡大

**なぜこれが特異なのか**:
- 従来の研修は「講師が教える」→「受講者が覚える」→「忘れる」のサイクル
- Omoro Theoryは「受講者が自分で発見する」→「自分の言葉で持つ」→「自分で再現できる」のサイクル
- 外部依存から自己駆動への転換

### 3.4 理論的位置づけ（ポジショニング）

```
                    感情的（没入）
                         ↑
                         │
          フロー理論 ●   │   ● Omoro Theory
                         │      （唯一のポジション）
                         │
  組織主導 ←─────────────┼─────────────→ 個人主導
                         │
    従来型研修 ●         │         ● ストレングスベース
                         │
          コンピテンシー ●│ ● 自己決定理論
                         │
                         ↓
                    論理的（成果）
```

**Omoro Theoryの独自ポジション**:
- **横軸（個人主導）**: 個人の「おもろい」を起点とするため、右側に位置
- **縦軸（感情的）**: 感情を起点としつつ成果に着地させるため、上部に位置
- **唯一性**: 「個人主導 × 感情起点 × 成果着地」を同時に満たす理論は他にない

## 4. 総括：Omoro Theoryの理論的貢献

### 4.1 既存理論の統合

Omoro Theoryは、以下の既存理論の知見を統合的に活用する。

| 借用元 | 借用した概念 | Omoro Theoryでの発展 |
|--------|------------|---------------------|
| フロー理論 | 没入状態 | 偶発→意図的設計 |
| 自己決定理論 | 三欲求 | 充足→統合（Will/Can/Must） |
| 経験学習 | 体験→概念化 | 没入体験→パターン抽出→業務転用 |
| ストレングスベース | 強みを活かす | 強み→没入パターン（より具体的） |
| 組織開発 | 組織変革 | 個人改革との両輪化 |

### 4.2 独自の理論的貢献

| 貢献 | 内容 |
|------|------|
| **感情の設計化** | 「おもろい」という主観的感情を、「没入パターン」という設計可能な概念に変換 |
| **統合フレームワーク** | Will/Can/Mustの統合という、個人と組織の両方を満たす目標設定 |
| **両輪モデル** | 個人改革と組織改革の同時進行による、持続的変革の実現 |
| **正のフィードバックループ** | おもろい→成果→おもろいの自己強化サイクルの設計 |

### 4.3 一言での差別化表現

> **Omoro Theory（オモロ理論）とは、「おもろい」を「没入パターン」に変換し、個人と組織の両輪改革で「楽しみながら成果を出す」状態を意図的に設計する、B2B研修理論の新たなパラダイムである。**
