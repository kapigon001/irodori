# B2B研修分野における既存理論フレームワーク

## 調査概要
本ドキュメントは、B2B研修分野における既存の根本的な考え方・理論を網羅的に整理したものです。irodoriプロジェクト（中堅社員向け没入パターン研修）の理論的基盤として活用できます。

---

## 1. 学習理論系

### 1.1 成人学習理論（アンドラゴジー）

| 項目 | 内容 |
|------|------|
| **理論名** | アンドラゴジー（Andragogy） |
| **提唱者** | マルカム・ノールズ（Malcolm Knowles）、1970年代に体系化 |
| **核となる主張** | 成人の学習は子どもの学習（ペダゴジー）とは本質的に異なる。成人学習者は①自己概念が自律的、②豊富な経験を持つ、③学習へのレディネスが社会的役割に関連、④問題解決志向、⑤内発的動機が重要 |
| **B2B研修での適用** | 「なぜこの研修を受けるのか」の目的を事前に明示する／受講者の経験を活かした対話型設計／業務に直結する問題解決型のコンテンツ |
| **限界・課題** | 全ての成人が自律的学習者とは限らない／文化差の考慮不足／学習スタイルの個人差を捨象 |

### 1.2 経験学習モデル（コルブモデル）

| 項目 | 内容 |
|------|------|
| **理論名** | 経験学習サイクル（Experiential Learning Cycle） |
| **提唱者** | デイヴィッド・コルブ（David A. Kolb）、1984年 |
| **核となる主張** | 学習は4段階のサイクルで深まる：①具体的経験（Experience）→②内省的観察（Reflection）→③抽象的概念化（Conceptualization）→④能動的実験（Experimentation） |
| **B2B研修での適用** | ゲーム・ワーク（経験）→振り返り（内省）→パターン抽出（概念化）→業務での実践計画（実験）という研修設計／OJTとの連携 |
| **限界・課題** | サイクルが線形的すぎる批判／学習スタイルの4分類（収束型・発散型等）の実証性に疑問／文化的バイアス |

**irodoriとの関連**: irodoriの「体験起点の気づき」は経験学習モデルと強く整合。ゲーム体験→振り返り→パターン抽出の流れはコルブモデルそのもの。

### 1.3 変容学習理論

| 項目 | 内容 |
|------|------|
| **理論名** | 変容的学習（Transformative Learning） |
| **提唱者** | ジャック・メジロー（Jack Mezirow）、1978年 |
| **核となる主張** | 成人学習の本質は「準拠枠（ものの見方の習慣的枠組み）」の変容にある。危機やジレンマを通じて既存の前提を批判的に吟味し、新しい視点を獲得する「視点変容（Perspective Transformation）」が起こる |
| **10段階プロセス** | 当惑させるジレンマ→自己吟味→前提の批判的評価→他者の不満・変容プロセスの認識→新しい役割・行動の探求→行動計画の策定→新しい知識・スキルの習得→新しい役割の試行→新しい視点への自信構築→再統合 |
| **B2B研修での適用** | 「自分は○○タイプだ」という固定観念を揺さぶる体験設計／心理的安全の確保／省察的対話の促進 |
| **限界・課題** | 変容には時間がかかり1日研修では難しい／心理的安全の確保が前提／ファシリテーターの高いスキルが必要 |

### 1.4 成人発達理論

| 項目 | 内容 |
|------|------|
| **理論名** | 成人発達理論（Adult Development Theory） |
| **提唱者** | ロバート・キーガン（Robert Kegan）他、1980年代〜 |
| **核となる主張** | 成人の知性は3段階で発達：①環境順応型（約70%）→②自己主導型（約20%）→③自己変容型（約10%以下）。発達段階に応じた支援が必要 |
| **B2B研修での適用** | 中堅社員の多くは環境順応型から自己主導型への移行期にある／「自分の軸」を持つ支援が効果的 |
| **限界・課題** | 発達段階の測定が複雑／段階を「上下」で捉えるリスク／個人差が大きい |

## 2. 動機づけ理論系

### 2.1 自己決定理論（SDT）

| 項目 | 内容 |
|------|------|
| **理論名** | 自己決定理論（Self-Determination Theory: SDT） |
| **提唱者** | エドワード・デシ（Edward L. Deci）、リチャード・ライアン（Richard M. Ryan）、1985年 |
| **核となる主張** | 人間には3つの基本的心理欲求がある：①自律性（Autonomy）、②有能感（Competence）、③関係性（Relatedness）。これらが満たされると内発的動機づけが高まる |
| **5段階の動機づけ** | 外的調整（報酬/罰）→取り入れ的調整（罪悪感/自尊心）→同一化的調整（価値の内在化）→統合的調整（自己との一致）→内発的動機づけ |
| **B2B研修での適用** | 研修内での選択肢を増やす（自律性）／適切な難易度設定とフィードバック（有能感）／グループワークでの関係構築（関係性） |
| **限界・課題** | 文化差（集団主義文化での「自律性」の解釈）／短期研修での持続性／組織の報酬制度との整合性 |

**irodoriとの関連**: 「没入パターンを自分で発見し業務に活かす」設計は自律性を強く支援。体験を通じた有能感の獲得も可能。

### 2.2 フロー理論

| 項目 | 内容 |
|------|------|
| **理論名** | フロー理論（Flow Theory） |
| **提唱者** | ミハイ・チクセントミハイ（Mihaly Csikszentmihalyi）、1975年〜 |
| **核となる主張** | 「フロー」とは、活動に完全に没頭し、時間感覚を忘れ、自己意識が消失しながらもコントロール感がある最適経験状態。スキルと難易度のバランスが重要 |
| **フローの条件** | 明確な目標／即時フィードバック／スキルと挑戦のバランス（どちらも高いレベル）／行動と意識の融合 |
| **B2B研修での適用** | ゲームで意図的にフロー状態を誘発→その体験を言語化／参加者の習熟度に合わせた難易度調整／こまめなフィードバック設計 |
| **限界・課題** | フロー状態は意図的に作り出すのが難しい／個人差が大きい／業務環境での再現性 |

**irodoriとの関連**: irodoriの核心はまさにフロー理論。「没入パターン」はフロー状態に入りやすい条件の個人パターンを指す。

### 2.3 内発的動機づけ

| 項目 | 内容 |
|------|------|
| **理論名** | 内発的動機づけ（Intrinsic Motivation） |
| **提唱者** | エドワード・デシ（Edward L. Deci）他 |
| **核となる主張** | 外部からの報酬や罰ではなく、活動そのものへの興味・楽しさから生じる動機づけ。創造性、学習の質、持続性において外発的動機づけより優れる |
| **B2B研修での適用** | 「やらされ感」を排除した研修設計／好奇心を刺激するワーク／自己理解を深める機会の提供 |
| **限界・課題** | 全ての業務を内発的に動機づけることは不可能／外発的動機づけとの適切なバランスが必要 |

## 3. 組織開発系

### 3.1 学習する組織

| 項目 | 内容 |
|------|------|
| **理論名** | 学習する組織（Learning Organization） |
| **提唱者** | ピーター・センゲ（Peter Senge）、1990年『The Fifth Discipline』 |
| **5つのディシプリン** | ①自己マスタリー（Personal Mastery）②メンタルモデル（Mental Models）③共有ビジョン（Shared Vision）④チーム学習（Team Learning）⑤システム思考（Systems Thinking） |
| **核となる主張** | 組織が継続的に学習し、目的に向けて効果的に行動するための意識（アウェアネス）と能力を伸ばし続けることが競争優位の源泉 |
| **B2B研修での適用** | 個人の学びを組織の学びに接続する設計／チーム単位での参加推奨／研修後の組織内共有の仕組み |
| **限界・課題** | 組織文化の変革が前提で時間がかかる／経営層のコミットメントが必須／測定が困難 |

### 3.2 アクションラーニング

| 項目 | 内容 |
|------|------|
| **理論名** | アクションラーニング（Action Learning） |
| **提唱者** | レグ・レバンス（Reg Revans）、1930年代にオリジナル開発 |
| **核となる主張** | 「現実の課題を題材にして個人と組織が学習する手法」。問題解決とリフレクション（振り返り）を通じて、個人・チーム・組織の学習力を養成する |
| **6つの構成要素** | 問題（現実の課題）／グループ（4〜8名）／質問とリフレクション／行動への落とし込み／学習へのコミットメント／アクションラーニングコーチ |
| **B2B研修での適用** | 研修内で実際の業務課題を扱う／質問中心のファシリテーション／研修後の行動計画と実践 |
| **限界・課題** | 時間がかかる（通常複数回セッション）／ファシリテーターの専門性が必要／参加者の業務負荷 |

**補足**: アクションラーニングは「学習する組織のDNA」とも呼ばれ、両理論は密接に関連。

## 4. 能力開発モデル系

### 4.1 70:20:10モデル（ロミンガーの法則）

| 項目 | 内容 |
|------|------|
| **理論名** | 70:20:10モデル（Model for Learning and Development） |
| **提唱者** | マイケル・ロンバルド（Michael Lombardo）、ロバート・アイチンガー（Robert Eichinger）、1996年 |
| **核となる主張** | リーダーの学習源は、70%が業務経験（On the Job）、20%が他者との関係（薫陶・メンタリング）、10%が研修・訓練 |
| **B2B研修での適用** | 研修（10%）を経験学習（70%）と接続する設計／上司・同僚との関わり（20%）を促進する仕組み／研修後の実践計画を必須化 |
| **限界・課題** | 数字の根拠に疑義あり（実証研究の不足）／「研修は10%しか効果がない」という誤解を招きやすい／バランスが重要という本質の見落とし |

**irodoriとの関連**: 研修で「没入パターン」を発見し、それを業務経験（70%）で活かす設計は70:20:10と整合。

### 4.2 コンピテンシーモデル

| 項目 | 内容 |
|------|------|
| **理論名** | コンピテンシーモデル（Competency Model） |
| **提唱者** | デイヴィッド・マクレランド（David McClelland）、1970年代〜 |
| **核となる主張** | 高業績者に共通する「思考・行動特性」を抽出し、採用・育成・評価の基準とする。潜在的能力ではなく、実際に成果に結びついた行動を重視 |
| **B2B研修での適用** | 職種・階層別のコンピテンシーを明確化→育成すべきスキルギャップの可視化→研修設計 |
| **限界・課題** | モデル作成に時間とコストがかかる／環境変化に伴い陳腐化しやすい／「高業績者」の定義が曖昧になりやすい |

### 4.3 ストレングスベースアプローチ

| 項目 | 内容 |
|------|------|
| **理論名** | ストレングスベースアプローチ（Strength-Based Approach） |
| **代表ツール** | ストレングス・ファインダー（CliftonStrengths）、ギャラップ社、2001年〜 |
| **核となる主張** | 弱みを克服するよりも、強みを伸ばす方が高いパフォーマンスにつながる。34の資質（才能）から自分の強みを特定し、意図的に活かす |
| **B2B研修での適用** | 診断→強みの認識→チームでの相互理解→業務での活用計画／マネージャー向け「強みを活かすマネジメント」研修 |
| **限界・課題** | 診断ツールへの依存／弱みの無視はリスク／強みの過剰使用（強みが弱みになる）への対処 |

**irodoriとの関連**: 「没入パターン」は広義の「強み」と重なる。ただしストレングス・ファインダーが資質（才能）に焦点を当てるのに対し、irodoriは「行動パターン」に焦点を当てる点が差別化ポイント。

## 5. ポジティブ心理学系

### 5.1 PERMA理論（ウェルビーイング理論）

| 項目 | 内容 |
|------|------|
| **理論名** | PERMA理論 / ウェルビーイング理論 |
| **提唱者** | マーティン・セリグマン（Martin Seligman）、2011年 |
| **5つの要素** | **P**ositive Emotion（ポジティブ感情）、**E**ngagement（没頭・フロー状態）、**R**elationships（良好な人間関係）、**M**eaning（意味・意義）、**A**ccomplishment（達成） |
| **核となる主張** | 持続的幸福（フラーリッシング）は単なる快楽ではなく、5つの要素のバランスで実現される |
| **B2B研修での適用** | 研修自体でPERMAを体験させる設計／エンゲージメント向上施策の理論的基盤／「働きがい」と「働きやすさ」の両立 |
| **限界・課題** | 測定の難しさ／個人のウェルビーイングと組織の成果の関係は複雑／文化差への配慮 |

**irodoriとの関連**: PERMAの「E（Engagement=没頭）」がirodoriの中核概念「没入」と完全に一致。

### 5.2 ジョブ・クラフティング

| 項目 | 内容 |
|------|------|
| **理論名** | ジョブ・クラフティング（Job Crafting） |
| **提唱者** | エイミー・レズネスキー（Amy Wrzesniewski）、ジェーン・ダットン（Jane Dutton）、2001年 |
| **核となる主張** | 従業員が自ら仕事の境界を変化させ、仕事の意味ややりがいを高める行為。3つの次元：タスク境界の変更、関係性境界の変更、認知境界の変更 |
| **B2B研修での適用** | 「今の業務をどう捉え直すか」のワーク／没入パターンを業務に適用する際の具体的手法として活用 |
| **限界・課題** | 組織の許容範囲内でしか実行できない／全ての業務に適用できるわけではない |

**irodoriとの関連**: 「業務への転用設計」はジョブ・クラフティングの実践そのもの。没入パターンを業務に活かすとは、認知境界・タスク境界の変更を意味する。

## 7. irodoriとの理論的整合性マッピング

| irodoriの核 | 関連する既存理論 | 整合度 | 差別化ポイント |
|-------------|-----------------|--------|---------------|
| **没入行動の型化** | フロー理論、PERMA（E）、ストレングスベース | 高 | 「資質」ではなく「行動パターン」に焦点 |
| **業務への転用設計** | ジョブ・クラフティング、70:20:10、経験学習 | 高 | 研修内で具体的なアクションまで落とし込む |
| **体験起点の気づき** | 経験学習モデル、アンドラゴジー、変容学習 | 高 | ゲームを没入誘発の手段として明確に位置づけ |

### 理論的基盤としての推奨

irodori研修の理論的基盤として、以下の組み合わせを推奨：

1. **中核理論**: フロー理論（没入の科学的根拠）
2. **学習設計**: 経験学習モデル（体験→振り返り→概念化→実践）
3. **動機づけ**: 自己決定理論（自律性・有能感・関係性の充足）
4. **業務転用**: ジョブ・クラフティング（仕事の再設計）
5. **組織接続**: 70:20:10モデル（研修と業務の接続）
