# チラシ品質チェックリスト

## 1. デザイン・レイアウト

### 情報設計
- [ ] 余白が十分にあるか（詰め込みすぎていないか）
- [ ] 視線誘導（Zの法則）に沿った配置になっているか
- [ ] 最重要情報（キャッチコピー）が左上にあるか
- [ ] CTA（連絡先）が右下にあるか
- [ ] 関連情報がグループ化（近接）されているか
- [ ] 要素の端が揃っている（整列）か

### 配色
- [ ] 使用色は3色以内か（ベース70%・メイン25%・アクセント5%）
- [ ] コントラストが十分か（読みやすさ）
- [ ] ブランドカラーとの整合性があるか
- [ ] アクセントカラーがCTAに使われているか

### タイポグラフィ
- [ ] フォント種類は2種類以内か
- [ ] 見出しと本文のサイズ差（ジャンプ率）が適切か
- [ ] 本文の文字サイズは読みやすいか（10pt以上推奨）
- [ ] 行間・字間は適切か

### 画像・図版
- [ ] 画像解像度は十分か（印刷350dpi以上）
- [ ] 図版は内容を補完しているか（装飾だけになっていないか）
- [ ] 人物画像の視線は適切な方向を向いているか

---

## 2. コピー・コンテンツ

### キャッチコピー
- [ ] 3秒で内容が伝わるか
- [ ] ターゲットの「痛み」に刺さっているか
- [ ] 機能ではなくベネフィットを語っているか
- [ ] 抽象的すぎないか（具体性があるか）

### ボディコピー
- [ ] 課題→原因→解決→根拠→行動の流れになっているか
- [ ] 1文が長すぎないか（40字以内推奨）
- [ ] 専門用語を使いすぎていないか
- [ ] 「私たち」ではなく「あなた」視点で書かれているか

### CTA（行動喚起）
- [ ] 何をすればいいか明確か
- [ ] ハードルを下げる言葉があるか（無料、30分、簡単）
- [ ] 連絡先情報は正確か
- [ ] QRコードは機能するか

---

## 3. 心理トリガー

### 必須トリガー
- [ ] **共感**: ターゲットの「独り言」に寄り添っているか
- [ ] **損失回避**: 「放置するとこうなる」リスク訴求があるか
- [ ] **希望**: 「こう変われる」ポジティブな未来像があるか

### 推奨トリガー
- [ ] **社会的証明**: 導入実績、お客様の声があるか
- [ ] **権威性**: 監修者、資格、メディア掲載があるか
- [ ] **緊急性**: 「今だけ」「先着〇名」があるか
- [ ] **具体性**: 数字、固有名詞が使われているか

---

## 4. 法務・コンプライアンス

### NG表現チェック
- [ ] 「No.1」「最高」「絶対」など根拠のない最上級表現はないか
- [ ] 効果を断定する表現はないか（「必ず変わる」など）
- [ ] 競合他社を貶める表現はないか
- [ ] 著作権・商標権を侵害する素材はないか

### 必須記載
- [ ] 会社名・屋号は正確か
- [ ] 連絡先（電話、メール、URL）は正確か
- [ ] 価格表示がある場合、税込/税別は明記されているか

---

## 5. 印刷・入稿

### データ仕様
- [ ] サイズは正確か（A4: 210mm×297mm）
- [ ] 塗り足しは設定されているか（3mm推奨）
- [ ] カラーモードはCMYKか
- [ ] フォントはアウトライン化されているか
- [ ] リンク画像は埋め込まれているか

### 校正
- [ ] 誤字脱字はないか
- [ ] 電話番号・URLは実際にアクセスして確認したか
- [ ] QRコードは実機でスキャンして確認したか
- [ ] 第三者にチェックしてもらったか

---

## 6. irodori固有チェック

### 理念との整合性
- [ ] 「没入パターン発見→業務転用」の価値が伝わっているか
- [ ] 「体験型研修」であることが伝わっているか
- [ ] 自己啓発・スピリチュアル表現は使われていないか

### トーン
- [ ] 「売り込み」ではなく「課題解決の提案」になっているか
- [ ] 経営者の「独り言」への共感から入っているか
- [ ] 上から目線ではなく、伴走者としての姿勢が感じられるか

### 実績がない場合の表現
- [ ] 「期待できる変化」として断定を避けているか
- [ ] ストーリー形式の場合「想定される変化」と明記しているか
- [ ] 仮置き・推定の情報には注釈があるか
