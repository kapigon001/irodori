# BtoB研修チラシ コピーライティングガイド

## 1. ベネフィット変換の技法

### メリットとベネフィットの違い

| 区分 | 定義 | 例（研修） |
|------|------|-----------|
| **メリット（特徴）** | 商品・サービスの機能や強み | 「没入パターン発見ワーク」 |
| **ベネフィット（価値）** | 顧客が得られる利益・未来 | 「部下への接し方に迷わなくなる」 |

### ベネフィットの3分類

| タイプ | 説明 | 研修での例 |
|--------|------|-----------|
| **機能的** | 便利・効率・時短 | 「会議が30分で終わるようになる」 |
| **情緒的** | 安心・自信・解放感 | 「"あいつどうしよう"と悩まなくなる」 |
| **自己表現的** | 理想の自分・ステータス | 「"あの社長はマネジメントがうまい"と言われる」 |

### 変換の手順

1. **特徴を書き出す**: 研修で何をやるか
2. **So What?を問う**: だから顧客にとって何がいいのか
3. **感情を加える**: どんな気持ちになれるのか
4. **具体化する**: いつ・どこで・誰が・どう変わるのか

```
【変換例】
特徴: 「没入パターン発見ワーク」
↓ So What?
「自分の強みが分かる」
↓ So What?
「強みを活かした仕事の任せ方ができる」
↓ 感情
「"この仕事、向いてるかも"と部下が思う瞬間を作れる」
↓ 具体化
「"やらされ感"で働いていた部下が、自分から動き出す」
```

## 2. キャッチコピーのパターン

### パターン一覧

| パターン | 構造 | 例 |
|---------|------|-----|
| **ターゲット呼びかけ** | 〇〇な方へ | 「"部下が動かない"と悩む経営者へ」 |
| **独り言共感** | 「〇〇」 | 「"あいつら、昔はもっとギラついてた"」 |
| **数字・具体性** | 〇〇%、〇〇人 | 「導入企業の92%が"変化を実感"」 |
| **Before/After** | 〇〇から〇〇へ | 「"どうせ俺なんか"から"俺、これ得意なんで"へ」 |
| **問いかけ** | 〇〇ですか？ | 「その社員、本当に"やる気がない"のですか？」 |
| **結論提示** | 〇〇する方法 | 「"火が消えた社員"に再点火する方法」 |
| **逆説・意外性** | 〇〇ではなく〇〇 | 「"やる気を出せ"では、人は動かない」 |

### BtoB研修チラシ推奨パターン

**第1推奨: 独り言共感型**
- 経営者が「これ、俺のことだ」と思う
- 感情に刺さるので、読み進めてもらえる
- 例: 「"あいつら、昔はもっとギラついてた"」

**第2推奨: 問いかけ型**
- 思い込みを揺さぶる
- 続きを読みたくなる
- 例: 「その社員、本当に"やる気がない"のですか？」

**第3推奨: Before/After型**
- 変化のイメージが湧く
- 研修の効果が伝わる
- 例: 「"指示待ち"が"自走"に変わる」

## 4. 経営者の「独り言」活用法

### 収集した独り言の使い方

| 使用場所 | 使い方 | 例 |
|---------|--------|-----|
| **ヘッドライン** | そのまま引用 | 「"あいつら、昔はもっとギラついてた"」 |
| **症状チェック** | リスト化 | □ 〇〇と思ったことがある |
| **共感コピー** | 言い換え | 「あの社員、昔はもっと...」と思い浮かぶ顔がありませんか？ |
| **リスク訴求** | 深刻化 | その状態が続くと、〇〇になります |

### 独り言→コピー変換例

```
【独り言】
「技術はあるのに、"判断"ができない。いつまで俺に聞くんだ」

【ヘッドライン案】
「いつまで俺に聞くんだ」——ベテランの"判断力"を取り戻す研修

【症状チェック案】
□ 技術はあるのに、判断を社長に委ねる社員がいる

【共感コピー案】
「技術はある。経験もある。でも、なぜ自分で判断しないのか？」
そう思う社員がいるなら、この研修が役に立つかもしれません。

【リスク訴求案】
判断を委ねる習慣が続くと、いつまでも社長の時間が奪われ続けます。
```

## 5. 避けるべき表現

### NG表現リスト

| カテゴリ | NG例 | 理由 |
|---------|------|------|
| **断定** | 「必ず変わります」 | 効果を保証できない |
| **最上級** | 「最高の研修」「No.1」 | 根拠がないと誇大広告 |
| **抽象語** | 「素晴らしい」「感動」 | 具体性がない |
| **自己啓発** | 「目覚め」「覚醒」「本当の自分」 | irodori理念に反する |
| **売り込み** | 「今すぐお申し込みを！」 | BtoBには不向き |

### OK表現への言い換え

| NG | OK |
|-----|-----|
| 「必ず変わります」 | 「〇〇という変化が期待できます」 |
| 「最高の研修」 | 「〇〇に特化した研修」 |
| 「感動の体験」 | 「"これか"という気づきが得られる体験」 |
| 「本当の自分に出会う」 | 「没入できる条件を言語化する」 |
| 「今すぐお申し込みを！」 | 「まずは30分の無料相談から」 |

## 6. マイクロコピーの活用

### CTAボタン周辺

| 要素 | 役割 | 例 |
|------|------|-----|
| **ハードル下げ** | 行動の障壁を下げる | 「無料」「30分」「1分で完了」 |
| **安心感** | 不安を払拭 | 「しつこい営業はしません」 |
| **次のステップ明示** | 何が起こるか伝える | 「担当者から24時間以内にご連絡します」 |

### 例文

```
【CTA】
まずは30分の無料相談から

【マイクロコピー】
- 「この段階で費用は一切かかりません」
- 「御社の課題を整理するだけでもOKです」
- 「しつこい営業はいたしません」
```
