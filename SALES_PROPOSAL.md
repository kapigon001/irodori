## エグゼクティブサマリー

**課題**: 年間の研修投資が行動変容に繋がらない
**解決策**: ゲーム体験から気づきを得て、自ら行動を変える研修
**効果**: 研修後も自己再現可能な行動変容を実現

---

## 1. 貴社の課題に寄り添う

### よくある研修の悩み

| 悩み | 背景 |
|------|------|
| 「研修直後は意識が高まるが、1週間で元に戻る」 | 座学中心で「自分ごと」にならない |
| 「受講者が受け身で、やらされ感がある」 | 教える側・教わる側の構造が主体性を奪う |
| 「研修と現場が繋がらない」 | 抽象的な学びが具体的な業務に落ちない |
| 「個人の強みが活かされていない」 | 画一的なスキル習得が前提になっている |

### 問題の本質

従来の研修は**「知識を注入すれば行動が変わる」**という前提に立っています。

しかし、人は**「教えられたこと」より「自分で気づいたこと」で変わります**。

---

## 2. 私たちの解決策

### 体験起点の研修プログラム

**ゲーム体験 → 気づき → 自己分析 → 行動変容**

受講者が自ら体験し、自ら気づき、自ら変わる研修です。

### 研修の流れ（1日6時間）

```
┌─────────────────────────────────────────────────────────┐
│  オリエンテーション（30分）                              │

├─────────────────────────────────────────────────────────┤
│  アイスブレイク（30分）                                  │
│  └ 連想ゲーム等で場を温め、心理的安全性を確保            │
├─────────────────────────────────────────────────────────┤
│  メインゲーム体験（90分）                                │
│  └ 実在ゲームで没頭・試行錯誤を繰り返す                  │
├─────────────────────────────────────────────────────────┤
│  昼休憩（60分）                                          │
├─────────────────────────────────────────────────────────┤
│  振り返り・気づきの言語化（30分）                        │
│  └ 問いへの回答を共有し、1つのメッセージを伝える         │

│  自己分析・他己分析（60分）                              │

```

| 感情 | 退屈・義務感 | 楽しい・夢中になる |

## 3. 理論的裏付け

### 本研修を支える理論体系

私たちの研修は、**行動科学・心理学の知見を統合した独自の理論体系**に基づいています。

#### 3つの学術的基盤

| 理論 | 知見 | 研修への活用 |
|------|------|-------------|
| **フロー理論**（Csikszentmihalyi） | 没頭状態では最高のパフォーマンスが発揮される | ゲームで意図的に没頭状態を生成する |
| **経験学習理論**（Kolb） | 人は体験→内省→概念化→実践のサイクルで学ぶ | 体験から始め、内省を経て業務に接続する |
| **自己決定理論**（Deci & Ryan） | 自律性・有能感・関係性が内発的動機を高める | 自ら発見し、自ら選ぶ設計にする |

#### 私たちの設計思想

```
【従来の研修】
  知識注入 → スキル習得 → 行動変容（期待）
  ※実際は行動変容が起きにくい

【私たちの研修】
  ゲーム体験 → 気づき → 個性の言語化 → 没入パターンの型化 → 業務接続
  ※自分で発見したことは行動に繋がる
```

### 「没入パターン」という考え方

人には**「この条件が揃うと夢中になれる」というパターン**があります。

本研修では、ゲーム体験を通じてこのパターンを発見し、言語化し、業務に転用します。

| 要素 | 内容 | 例 |
|------|------|-----|
| **状況** | どんな場面で夢中になるか | 「チームで協力して難題に挑むとき」 |
| **条件** | 何があると夢中になれるか | 「明確なゴールと裁量がある」 |
| **行動** | 夢中なとき何をしているか | 「情報を整理して全体像を示す」 |
| **再現条件** | 業務でどう再現するか | 「会議の冒頭で論点整理を担当する」 |

### 4-1. 強み再発見研修（中堅社員向け）

**対象**: 入社5年目以上の中堅社員
**課題**: 自分の強みが分からない、マンネリ感がある
**1メッセージ**: 「成果は試行錯誤の先にある」

#### 使用ゲーム：ヌメロン

相手の3桁の数字を、質問と推理で当てるゲームです。

| 要素 | 内容 |
|------|------|
| ルール | 交互に数字を予想し、「◯EAT ◯BITE」のヒントを得る |
| 狙い | 仮説→検証→修正のサイクルを体感する |
| 気づき | 「最初から正解は分からない。試して、学んで、また試す」 |

#### タイムテーブル

| 時間 | 内容 |
|------|------|
| 9:00-9:30 | **オリエンテーション**：「あなたの強みは何ですか？」という問いを設定 |
| 9:30-10:00 | **アイスブレイク**：連想ゲーム（テーマ：仕事で大切にしていること） |
| 10:00-11:30 | **ヌメロン体験**：個人戦→チーム戦で没頭 |
| 11:30-12:30 | 昼休憩 |
| 12:30-13:00 | **振り返り**：問いへの回答を共有→「成果は試行錯誤の先にある」を伝える |
| 13:00-14:00 | **自己分析・他己分析**：ゲーム中の自分の役割・輝いた瞬間を相互フィードバック |
| 14:00-15:00 | **没入パターンの深掘り・型化**：「どんな条件で夢中になれるか」を言語化 |
| 15:00-16:00 | **業務接続・見える化**：会社/チーム方針と照らし、具体的な行動計画を作成 |

#### 成果物

- 個性プロファイルシート
- 没入パターンシート
- 業務接続アクションプラン

| 気づき | 「部下の強みも、本人には見えていない。伝えることが大切」 |

#### タイムテーブル

| 時間 | 内容 |
|------|------|
| 9:00-9:30 | **オリエンテーション**：「部下の強みを3つ言えますか？」という問いを設定 |
| 9:30-10:00 | **アイスブレイク**：共通点探しゲーム（ペアで意外な共通点を見つける） |
| 10:00-11:30 | **花火（HANABI）体験**：協力して高得点を目指す |
| 11:30-12:30 | 昼休憩 |
| 12:30-13:00 | **振り返り**：問いへの回答→「自分は他者の目を通して見える」を伝える |
| 13:00-14:00 | **自己分析・他己分析**：ゲーム中に受けたヒントから自分の傾向を分析 |
| 14:00-15:00 | **没入パターンの深掘り・型化**：自分とメンバーの没入パターンを整理 |
| 15:00-16:00 | **業務接続・見える化**：メンバーへのフィードバック計画を作成 |

#### 成果物

- 自己認識シート（他者からの視点）
- メンバー没入パターン仮説シート
- フィードバック実行計画
