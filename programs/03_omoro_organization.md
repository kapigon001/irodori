# おもろい組織づくりワークショップ

## 概要

### 研修の目的
個人と組織の両輪で「おもろい」状態---すなわち**個性が外から見えて輝いている状態**---を意図的に設計し、持続的な成果創出を実現する組織文化を構築する。

### 対象
中堅社員を核とした部門横断チーム

### 期待成果
1. **個人レベル**: 自分の「没入パターン」を言語化し、業務で再現できるようになる
2. **チームレベル**: メンバーの個性を相互理解し、活かし合う関係性を構築する
3. **組織レベル**: Will/Can/Mustが統合された状態を設計し、「おもろい組織」の土台を築く

### 基本情報
- **時間**: 2日間（6時間 x 2）
- **人数**: 20-30名（5-6名 x 4-5チーム）
- **形式**: 体験学習中心（講義2割、ワーク8割）

---

## Omoro Theoryの核心（本研修の理論的基盤）

| 核心概念 | 内容 |
|---------|------|
| **「おもろい」の定義** | 個性が外から見えて輝いている状態（感情ではなく観察可能な状態） |
| **改革の両輪** | 個人改革 x 組織改革を同時に回す |
| **Will/Can/Must統合** | やりたい・できる・求められるの三要素が重なる状態を設計する |
| **手段** | ゲーム・対話・心理的安全性 |

## プログラム構成

### Day1: 「おもろい」個人を発見する（個人改革）

#### オープニング（30分）9:00-9:30
- **目的**: 研修の全体像を理解し、安心して参加できる場をつくる
- **内容**:
  - ファシリテーター自己紹介
  - 研修の目的と2日間の流れ説明
  - グランドルール設定（心理的安全性の確保）
  - チーム編成と自己紹介ワーク
- **ワーク**: 1分間自己紹介「最近"夢中になった"こと」
- **成果物**: チーム編成完了、グランドルール合意

#### セッション1: 「おもろい」とは何か（60分）9:30-10:30
- **目的**: 「おもろい」の定義を体感的に理解する
- **内容**:
  - 従来の「おもろい=楽しい」との違いを解説
  - 「個性が外から見えて輝いている状態」の具体例紹介
  - 動画視聴：没入状態で働く人々の姿
  - グループディスカッション：「輝いている人」の共通点
- **ワーク**: ペアで「あなたが輝いていた瞬間」を聴き合う（10分 x 2）
- **成果物**: 「おもろい」状態の個人的定義（ワークシート1）

**休憩（15分）10:30-10:45**

#### セッション2: 没入パターン発見ゲーム（90分）10:45-12:15
- **目的**: ゲーム体験を通じて、自分の没入パターンを発見する
- **内容**:
  - ゲーム体験（3種類のミニゲームをローテーション）
    1. 戦略型ゲーム（ボードゲーム/カードゲーム）
    2. 創造型ゲーム（レゴ/粘土で課題解決）
    3. コミュニケーション型ゲーム（即興演劇/ディベート）
  - 各ゲーム後に「観察者」からフィードバック
  - 「何が、どうあったから輝けたか」を言語化
- **ワーク**:
  - 各ゲーム15分 + 振り返り10分
  - 他者観察シートへの記入
- **成果物**: 他者観察シート（相互フィードバック用）

**昼食（60分）12:15-13:15**

#### セッション3: 没入パターンの言語化（90分）13:15-14:45
- **目的**: ゲーム体験から没入パターンを抽出し、型化する
- **内容**:
  - フィードバック交換：他者から見た「あなたが輝いていた瞬間」
  - 自己分析：過去の業務・プライベートでの没入体験の振り返り
  - 没入パターンの構造化（SCBR法）
    - Situation（状況）
    - Condition（条件）
    - Behavior（行動）
    - Result（結果）
- **ワーク**: 没入パターンシート作成
- **成果物**: 没入パターンシート（個人用）

**休憩（15分）14:45-15:00**

#### セッション4: 個性プロファイル作成（75分）15:00-16:15
- **目的**: 没入パターンを「個性」として統合し、言語化する
- **内容**:
  - 没入パターンから見える「人間的本質」の抽出
  - 他者からのフィードバックとの統合
  - 個性プロファイルの作成
    - 「私が輝く条件」
    - 「私の没入トリガー」
    - 「私がチームに貢献できること」
- **ワーク**: 個性プロファイル作成 + ペアでブラッシュアップ
- **成果物**: 個性プロファイル（ドラフト版）

#### Day1クロージング（45分）16:15-17:00
- **目的**: Day1の学びを統合し、Day2への橋渡しをする
- **内容**:
  - チーム内で個性プロファイル共有（3分/人）
  - Day1の気づきを3つのキーワードで表現
  - Day2予告：「個人→組織」への展開
  - 宿題説明：「私の没入パターンが活きた業務場面」を1つ思い出す
- **ワーク**: チーム共有 + 全体発表（代表者）
- **成果物**: Day1振り返りシート

---

### Day2: 「おもろい」組織を設計する（組織改革）

#### Day2オープニング（30分）9:00-9:30
- **目的**: Day1の復習とDay2の準備
- **内容**:
  - 宿題共有：「私の没入パターンが活きた業務場面」
  - Day1の学びの振り返り
  - Day2の目標設定：「個人」から「組織」へ
- **ワーク**: ペアで宿題共有（5分 x 2）
- **成果物**: 宿題共有からの追加気づき

#### セッション5: 改革の両輪モデル（60分）9:30-10:30
- **目的**: 個人改革と組織改革の連動を理解する
- **内容**:
  - 両輪モデルの解説
    - 個人改革：個性の抽出→没入行動の特定→業務への統合
    - 組織改革：理念浸透→環境設計→状態目標
  - なぜ両輪が必要なのか（事例紹介）
  - 「個人が輝いても、組織が活かさなければ成果にならない」
  - 「組織が整えても、個人が自分を知らなければ活用できない」
- **ワーク**: 自組織の「両輪」現状診断（個人ワーク→チーム共有）
- **成果物**: 組織現状診断シート

**休憩（15分）10:30-10:45**

#### セッション6: Will/Can/Must統合ワークショップ（90分）10:45-12:15
- **目的**: 三要素が重なる「最適状態」を個人・チームレベルで設計する
- **内容**:
  - Will/Can/Must統合フレームワークの解説
    - Will x Can のみ：楽しいが求められない
    - Will x Must のみ：やりたいが できない
    - Can x Must のみ：できるがやりたくない
    - Will x Can x Must：最適状態（おもろい x 成果）
  - 個人の統合マップ作成
  - チームでの統合可能性探索
- **ワーク**:
  - 個人統合マップ作成（30分）
  - チーム統合セッション（40分）
    - メンバーのWill/Can/Mustを共有
    - チームとして統合できる領域を発見
- **成果物**: 個人統合マップ、チーム統合マップ

**昼食（60分）12:15-13:15**

#### セッション7: おもろい組織の環境設計（75分）13:15-14:30
- **目的**: 個性が活きる組織環境を具体的に設計する
- **内容**:
  - 心理的安全性の重要性
  - 環境設計の3要素
    1. 会議の仕方（個性が発揮できる場づくり）
    2. 役割・配置（没入パターンを活かす配置）
    3. フィードバック（「輝き」を認め合う文化）
  - 自組織の環境改善アイデア出し
- **ワーク**:
  - 「おもろい会議」のルール設計（チーム）
  - 環境改善アイデアブレスト（付箋ワーク）
- **成果物**: おもろい会議ルール、環境改善アイデアリスト

**休憩（15分）14:30-14:45**

#### セッション8: アクションプラン策定（75分）14:45-16:00
- **目的**: 学びを具体的な行動に落とし込む
- **内容**:
  - 個人アクションプラン作成
    - 「私の没入パターンを業務で活かす具体的行動」
    - 「チームメンバーの個性を活かすために私ができること」
  - チームアクションプラン作成
    - 「おもろい組織」に向けた具体的施策
    - 3ヶ月後の目標状態
- **ワーク**:
  - 個人アクションプラン作成（20分）
  - チームアクションプラン作成（30分）
  - ペアコミットメント（相互宣言）
- **成果物**: 個人アクションプラン、チームアクションプラン

#### クロージング（60分）16:00-17:00
- **目的**: 2日間の学びを統合し、行動へのコミットメントを高める
- **内容**:
  - チームアクションプラン発表（5分/チーム）
  - 全体振り返り：「おもろい」の再定義
  - 状態KPIの紹介
    - 没入時間率
    - 没入パターン活用率
    - 個性発揮実感度
  - 宣言ワーク：「私は○○することで、おもろい組織に貢献する」
  - ファシリテーターからのメッセージ
- **ワーク**: 全体発表、宣言ワーク
- **成果物**: チームアクションプラン発表資料、個人宣言カード

## ファシリテーターへの注意点

### 心理的安全性の確保

1. **ジャッジしない姿勢の徹底**
   - 参加者の発言を評価しない
   - 「正解」を求めない問いかけをする
   - 沈黙を恐れず、待つ姿勢を持つ

2. **グランドルールの遵守**
   - 研修冒頭で合意したルールを都度確認
   - 否定的な言動が見られたら即座に介入
   - 「失敗」を「発見」として捉える文化づくり

3. **個性の多様性を尊重**
   - 内向的な参加者への配慮（書く時間の確保）
   - 発言量の偏りへの介入
   - 「輝き方」は人それぞれであることを強調

### 「おもろい」の定義の正確な伝達

1. **誤解を防ぐ説明**
   - 「おもろい=楽しい」ではないことを明確に
   - 「観察可能な状態」であることを強調
   - 具体例を多用して体感的理解を促す

2. **言い換えの準備**
   - 「個性が輝いている状態」
   - 「その人らしさが発揮されている状態」
   - 「没入して成果を出している状態」

### ワーク進行のポイント

1. **体験→振り返り→概念化の順序を守る**
   - 理論説明は体験の後
   - 問いかけで気づきを引き出す
   - 講師が「答え」を言わない

2. **時間管理**
   - ワーク時間は厳守
   - 延長する場合は全体スケジュールを調整
   - 深い対話が生まれている場合は柔軟に対応

3. **チームダイナミクスの観察**
   - 発言の偏りをチェック
   - チーム間の進捗差に対応
   - 必要に応じてチーム再編成

### 避けるべきこと

| 避けるべき行動 | 理由 | 代替行動 |
|--------------|------|---------|
| 理論から入る | 体験起点の原則に反する | ゲーム・ワークから始める |
| 「正解」を提示する | 自己発見を阻害する | 問いかけで引き出す |
| 参加者を比較する | 心理的安全性を損なう | 個々の「輝き」を認める |
| 時間を大幅に超過する | 全体設計が崩れる | タイムキープを徹底 |
| 自己啓発的表現を使う | irodoriの品質基準に反する | 観察可能な行動・状態で表現 |
