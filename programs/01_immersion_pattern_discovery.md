# 没入パターン発見研修

**irodori Training Program 01**

---

## 概要

### 研修の目的
中堅社員が自分の「おもろい」状態---すなわち**個性が外から見えて輝いている状態**---を発見し、それを「没入パターン」として言語化することで、業務において意図的に再現できるようになる。

### 対象者
- 入社5-10年目の中堅社員
- 自分の強みや働き方に迷いがある方
- 業務のマンネリ化を感じている方
- チームの中核として活躍が期待される方

### 期待成果
1. 自分が輝く条件（没入パターン）を3つ以上言語化できる
2. 没入パターンを現在の業務に転用するアクションプランを作成できる
3. Will/Can/Mustが重なる状態を自分で設計できるようになる

### 研修形式
- **時間**: 1日（6時間）
- **人数**: 15-20名
- **形式**: ゲーム体験 + グループ対話 + 個人ワーク

---

## プログラム構成

### セッション1: オープニング - 「おもろい」を知る（30分）

#### 目的
- 研修の目的と「おもろい」の定義を共有する
- 心理的安全性を確保し、参加者が自己開示しやすい場を作る

#### 内容
1. **チェックイン（10分）**
   - 隣の人と「最近、時間を忘れて没頭したこと」を30秒ずつ共有
   - 全体で2-3名が発表

2. **「おもろい」の定義共有（10分）**
   - 「おもろい」≠ 楽しい・嬉しいという感情
   - 「おもろい」= 個性が外から見えて輝いている状態（観察可能な状態）
   - 講師が具体例を提示（スポーツ選手、職人、身近な同僚など）

3. **研修のゴール説明（10分）**
   - 今日のゴール：自分の「おもろい」状態を引き起こす条件を発見する
   - 没入パターンシートとアクションプランの説明

#### 成果物
- なし（セッション2以降の準備）

#### ファシリテーターメモ
- 「おもろい」を「楽しい」と混同しないよう、明確に区別する
- 「外から見て輝いている」という観察可能性を強調する

---

### セッション2: ゲーム体験 - 没入状態を体験する（60分）

#### 目的
- 意図的に没入状態を生成し、自分が輝く瞬間を体験する
- 観察者が互いの「おもろい」状態を発見する

#### 内容
1. **ルール説明（5分）**
   - 4-5名のグループに分かれる
   - 2種類のゲームを実施し、役割を交代

2. **ゲームA: タワービルディング（20分）**
   - 制限時間内に限られた素材で最も高いタワーを建てる
   - 1名は観察者として、メンバーの「輝いている瞬間」を記録

3. **ゲームB: ストーリーリレー（20分）**
   - 1人30秒ずつ話をつなげて物語を完成させる
   - 観察者は「誰がどの瞬間に輝いていたか」を記録

4. **観察フィードバック（15分）**
   - 観察者から各メンバーへ「あなたが輝いていた瞬間」をフィードバック
   - 本人の自覚と他者からの観察のギャップを確認

#### 成果物
- 観察シート（他者から見た自分の「輝いていた瞬間」メモ）

#### 使用ツール
- タワービルディング用素材（紙、テープ、ストローなど）
- 観察シート（A5用紙）
- タイマー

#### ファシリテーターメモ
- ゲームの勝敗よりも「没入体験」と「観察」が目的であることを強調
- 観察者には「何をしていたか」だけでなく「どう見えたか」を記録させる

---

### セッション3: 振り返り - 没入体験を言語化する（60分）

#### 目的
- ゲーム体験を経験学習サイクルに沿って振り返る
- 没入状態を引き起こした条件を言語化する

#### 内容
1. **個人振り返り（15分）**
   - ワークシートに沿って以下を記入
     - ゲーム中、最も没入していた瞬間はいつか
     - その時、何が起きていたか（状況）
     - なぜ没入できたと思うか（条件）
     - 他者からのフィードバックで気づいたこと

2. **ペア対話（20分）**
   - 2人1組で互いの振り返りを共有
   - 相手の話を聴きながら「それって、こういうことでは?」と言語化を助ける
   - 問いかけ例：「他にも同じように没入した経験はある?」

3. **グループ共有（15分）**
   - 4-5名のグループで、各自の「没入条件」を1分で発表
   - 共通点と違いを話し合う

4. **気づきの全体共有（10分）**
   - 各グループから1名が気づきを発表
   - ファシリテーターが共通パターンを整理

#### 成果物
- 振り返りワークシート（個人）
- 没入条件の言語化メモ

#### ファシリテーターメモ
- 「正解」を求めず、本人の言葉で表現することを促す
- 抽象的な表現は「具体的には?」「例えば?」で深掘りする

### セッション4: パターン抽出 - 没入パターンを型化する（60分）

#### 目的
- 過去の没入体験も含めて共通パターンを抽出する
- 没入パターンシートを完成させる

#### 内容
1. **過去の没入体験の棚卸し（15分）**
   - 仕事、趣味、学生時代など、過去に没入した体験を5つ書き出す
   - 各体験について「状況」「条件」「行動」「結果」を記入

2. **共通パターンの抽出（20分）**
   - 5つの体験を横に並べて共通点を探す
   - 共通する条件を3つ以上抽出
   - 例：「締め切りがある」「チームで協力」「新しいことに挑戦」

3. **没入パターンシートの作成（20分）**
   - 以下の形式で没入パターンを記述

   | 項目 | 内容 |
   |------|------|
   | パターン名 | 例：「チャレンジ型没入」 |
   | 状況（Situation） | どのような状況で輝くか |
   | 条件（Condition） | 何があれば輝けるか |
   | 行動（Behavior） | どのような行動を取っているか |
   | 結果（Result） | どのような成果・状態が生まれるか |

4. **ペアフィードバック（5分）**
   - 隣の人と没入パターンシートを共有
   - 「自分らしさが出ているか」を確認

#### 成果物
- 没入パターンシート（3パターン以上）

#### 使用ツール
- 没入体験棚卸しシート
- 没入パターンシート（A4用紙）

#### ファシリテーターメモ
- パターンは人それぞれ異なることを強調
- 「正しいパターン」ではなく「自分のパターン」を発見することが目的

---

### セッション5: 業務転用 - Will/Can/Must統合設計（60分）

#### 目的
- 没入パターンを現在の業務に転用する方法を設計する
- Will/Can/Mustが重なる状態を意図的に作り出す

#### 内容
1. **Will/Can/Mustフレームワーク説明（10分）**
   - Will：やりたいこと（内発的動機）
   - Can：できること（有能感）
   - Must：求められること（組織要求）
   - 3つが重なる状態 = 「おもろい × 成果」の最適状態

2. **現状分析ワーク（15分）**
   - 現在の業務を3つ書き出す
   - 各業務についてWill/Can/Mustの重なり具合を自己評価
   - 重なりが薄い部分を特定

3. **没入パターン × 業務マッチング（20分）**
   - 没入パターンシートを見ながら、各業務との接点を探す
   - 「この没入パターンを使えば、この業務がもっとおもろくなる」を発見
   - 具体的なアクションに落とし込む

4. **アクションプラン作成（15分）**
   - 以下の形式でアクションプランを作成

   | 項目 | 内容 |
   |------|------|
   | 対象業務 | 転用する業務 |
   | 活用する没入パターン | 使うパターン名 |
   | 具体的アクション | 何をするか（観察可能な行動） |
   | 実施時期 | いつ実行するか |
   | 成功の判断基準 | 何が起きたら成功か |

#### 成果物
- Will/Can/Must分析シート
- 業務転用アクションプラン（2つ以上）

#### 使用ツール
- Will/Can/Must分析シート
- アクションプランシート

#### ファシリテーターメモ
- アクションは「具体的で観察可能な行動」になっているか確認
- 「明日からできること」を1つ必ず含める

---

### セッション6: クロージング - コミットメントと宣言（30分）

#### 目的
- 研修の学びを統合し、行動へのコミットメントを高める
- 参加者同士で宣言することで実行確率を上げる
- **組織改革との接続**：上司・チームとの共有方法を設計する

#### 内容
1. **学びの統合（10分）**
   - 今日の研修で得た最も重要な気づきを1つ選ぶ
   - 「私の没入パターンは〇〇」「明日から〇〇する」を言語化

2. **組織への接続ワーク（10分）** ※新規追加
   - 上司・チームへの共有方法を設計
   - 「没入パターンを上司に伝える一言」を作成
   - 「チームメンバーに協力してもらいたいこと」を明確化

3. **宣言タイム（8分）**
   - グループ内で1人ずつ宣言
   - 「私は〇〇という没入パターンを活かして、〇〇します」
   - メンバーは拍手とポジティブフィードバック

4. **チェックアウト（2分）**
   - 研修全体を通じた感想を一言ずつ共有
   - ファシリテーターからのメッセージ

#### 成果物
- 宣言カード（持ち帰り用）

#### ファシリテーターメモ
- 宣言は強制せず、本人が言いたいことを尊重
- 「おもろい」状態で終われるよう、ポジティブな雰囲気で締める

## ファシリテーターへの注意点

### 心理的安全性の確保
1. **批判・否定をしない場づくり**
   - 「正解はない」「あなたのパターンはあなただけのもの」を繰り返し伝える
   - 他者の発言を否定する言動があれば、すぐに介入

2. **自己開示は強制しない**
   - 共有したくないことは共有しなくてよいと伝える
   - パスの選択肢を常に用意

3. **観察フィードバックのルール**
   - ポジティブな観察のみを伝える
   - 「輝いていた」「没入していた」という表現を使う

### 言語化の支援
1. **抽象的な表現の具体化**
   - 「楽しかった」→「何が楽しかった?」
   - 「頑張った」→「具体的にどんな行動をした?」

2. **パターン発見の補助**
   - 「他にも同じような体験はある?」
   - 「共通しているのは何だと思う?」

3. **業務転用の現実性確認**
   - 「明日からできる?」
   - 「具体的にいつ、何をする?」

### 避けるべきこと
- 自己啓発的・スピリチュアルな表現
- 「あなたの強みは〇〇です」と断定する
- 診断ツールの結果に依存する
- 座学や講義に時間を使いすぎる

## 付録：Omoro Theory との対応

本研修プログラムは、Omoro Theoryの核心を以下のように反映している。

| Omoro Theoryの要素 | 本研修での実現方法 |
|-------------------|-------------------|
| 「おもろい」= 個性が外から見えて輝いている状態 | セッション2で他者観察を通じて「輝いている瞬間」を発見 |
| 没入パターンの発見・型化 | セッション3-4で過去体験から共通パターンを抽出 |
| 個性の言語化・見える化 | 没入パターンシートで具体的に言語化 |
| Will/Can/Must統合 | セッション5で3要素の統合を設計 |
| 体験起点の気づき | 座学ではなくゲーム体験から始める設計 |
| 心理的安全性 | 全セッションで批判しない場づくりを徹底 |
