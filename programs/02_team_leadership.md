# チームの個性を活かすリーダーシップ研修

## 概要

### 研修の目的
マネージャー・チームリーダーが、Omoro Theoryに基づいてメンバーの「おもろい」状態を引き出し、チーム全体の成果に繋げるスキルを習得する。

### 対象者
- マネージャー・チームリーダー（部下3名以上）
- チーム運営に課題を感じている管理職
- メンバーのモチベーション向上を目指すリーダー

### 期待成果
1. 「おもろい」状態の定義と観察方法を理解する
2. メンバーの没入パターンを発見・言語化できるようになる
3. 組織改革の3ステップ（理念浸透・環境設計・状態目標）を実践できる
4. チーム全員が「おもろい」状態で成果を出す環境を設計できる

### 研修仕様
| 項目 | 内容 |
|------|------|
| 時間 | 1日（6時間）※休憩含む |
| 人数 | 10-15名 |
| 形式 | ワークショップ形式（講義2割、ワーク8割） |
| 持ち物 | 筆記用具、事前課題シート |

---

## 事前課題

研修の効果を高めるため、受講者には以下の事前課題を依頼する。

### 事前課題1: メンバー観察シート
各メンバーについて、以下を観察・記録する（研修1週間前までに提出）

| 観察項目 | 記入内容 |
|----------|---------|
| メンバー名 | （仮名可） |
| 輝いていた瞬間 | 最近、そのメンバーが「輝いている」と感じた場面 |
| そのときの状況 | 何をしていたか、誰といたか、どのような条件だったか |
| 自分の働きかけ | その状況を作るために自分が行ったこと（あれば） |

### 事前課題2: チーム課題の言語化
現在のチームにおける課題を3つ挙げ、優先順位をつける

### セッション1: 「おもろい」とは何か（45分）9:15-10:00

- **目的**: 「おもろい」状態の定義を理解し、従来の管理手法との違いを認識する

- **内容**:
  1. **講義（15分）**: 「おもろい」の定義
     - 「おもろい」= 個性が外から見えて輝いている状態
     - 感情（楽しい・嬉しい）との違い
     - 観察可能な「状態」として捉える

  2. **ワーク1（20分）**: 「おもろい」状態の観察練習
     - ペアになり、事前課題で記録した「輝いていた瞬間」を共有
     - 聞き手は「何が、どうあったから輝いていたか」を質問で掘り下げる
     - 3つの要素を特定：状況、条件、行動

  3. **全体共有（10分）**: 気づきの共有

- **ワーク**:
  - ペアワーク：「輝いていた瞬間」の深掘り対話

- **成果物**:
  - 「おもろい」状態の3要素メモ（状況・条件・行動）

---

### セッション2: 没入パターンの発見（75分）10:00-11:15

- **目的**: ゲーム体験を通じて、メンバーの「没入パターン」を発見・言語化する手法を習得する

- **内容**:
  1. **講義（5分）**: 没入パターンとは
     - 没入パターン = 「この人が輝く条件」を型化したもの
     - フロー理論との接続：意図的に設計・再現する

  2. **ゲーム体験（25分）**: 「輝き発見ゲーム」 ※新規追加
     - チーム対抗の短時間ミニゲームを実施（例：ペーパータワー、連想ゲーム）
     - 各メンバーの役割を観察しながらゲームに参加
     - 「観察者シート」に他メンバーの輝いていた瞬間を記録
     - ゲーム後、「あなたは○○のとき輝いていた」を相互フィードバック

  3. **デモンストレーション（10分）**: 没入パターン抽出の実演
     - ゲーム中の観察をもとに、抽出プロセスを実演
     - 「状況→条件→行動→結果→再現条件」の流れを示す

  4. **ワーク2（25分）**: 没入パターン抽出ワーク
     - 3人1組でグループを作成
     - 1人が「自分が輝いた体験」を語る（5分）
     - 2人が質問で掘り下げ、没入パターンシートを記入（10分）
     - 役割を交代して全員分実施

  5. **振り返り（10分）**: ゲーム体験と抽出ワークからの気づきを共有

- **ワーク**:
  - ゲーム体験：「輝き発見ゲーム」
  - グループワーク：没入パターン抽出練習

- **成果物**:
  - 観察者シート（他メンバーの輝きフィードバック）
  - 自分自身の没入パターンシート（ドラフト版）

### セッション3: 両輪モデルの理解（45分）11:30-12:15

- **目的**: 個人改革と組織改革の「両輪」を同時に回す必要性を理解する

- **内容**:
  1. **講義（15分）**: 両輪モデルの解説
     - 個人改革の3ステップ：個性の抽出→没入行動の特定→業務への統合
     - 組織改革の3ステップ：理念浸透→環境設計→状態目標
     - なぜ「両輪」が必要か：片輪だけでは成果に繋がらない

  2. **ワーク3（20分）**: 両輪診断ワーク
     - 自チームの現状を「両輪診断シート」で評価
     - 個人改革の進捗度（5段階）
     - 組織改革の進捗度（5段階）
     - どちらの車輪が弱いかを特定

  3. **ペア共有（10分）**: 診断結果を隣の人と共有し、気づきを得る

- **ワーク**:
  - 個人ワーク：両輪診断シート記入

- **成果物**:

### セッション4: 組織改革の3ステップ実践（90分）13:00-14:30

- **目的**: 組織改革の3ステップ（理念浸透・環境設計・状態目標）を自チームに適用する

- **内容**:

#### Part1: 理念浸透（30分）
  1. **講義（10分）**: 理念浸透とは
     - 「個人の個性が活き、かつ成果に繋がる」理念・方針
     - 抽象的な理念ではなく、行動指針レベルまで落とし込む

  2. **ワーク4（20分）**: チーム理念の再設計
     - 自チームの現在の理念・方針を書き出す
     - 「個性が活きる」視点で追加・修正する
     - 「理念ステートメント（ドラフト）」を作成

#### Part2: 環境設計（30分）
  1. **講義（10分）**: 環境設計とは
     - 会議の仕方、人員配置、心理的安全性
     - 「個性を発揮できる環境」の具体的要素

  2. **ワーク5（20分）**: 環境設計マップ作成
     - 自チームの現在の環境を書き出す（会議、配置、コミュニケーション）
     - 「個性が活きる」視点で改善点を特定
     - 改善アクション3つを選定

#### Part3: 状態目標（30分）
  1. **講義（10分）**: 状態目標とは
     - 「全員が『おもろい』状態で成果を出す」をKPI化
     - 状態KPIの例：没入時間率、個性発揮実感度

  2. **ワーク6（20分）**: 状態KPI設計
     - 自チームの「状態KPI」を3つ設計
     - 測定方法と目標値を設定
     - 振り返りサイクルを決定

- **ワーク**:
  - 個人ワーク：各ステップのワークシート記入
  - グループ共有：各テーブルで1名発表

- **成果物**:
  - 理念ステートメント（ドラフト）
  - 環境設計マップ
  - 状態KPIシート

## ファシリテーターへの注意点

### 研修全体を通じて

1. **「おもろい」の定義を繰り返し確認する**
   - 参加者は「楽しい」「嬉しい」という感情と混同しやすい
   - 「個性が外から見えて輝いている状態」という定義を何度も確認する

2. **観察可能な行動レベルで語らせる**
   - 抽象的な表現（「モチベーションが高い」等）を具体的な行動に落とし込む
   - 「具体的には何をしていましたか？」「周囲から見てどう見えましたか？」と問う

3. **両輪の重要性を強調する**
   - 個人の没入パターン発見だけでは不十分
   - 組織改革との連動がなければ成果に繋がらないことを繰り返し伝える

### セッション別の注意点

| セッション | 注意点 |
|-----------|--------|
| セッション1 | 「おもろい」と「楽しい」の違いを丁寧に説明。混同している参加者には個別フォロー |
| セッション2 | 没入パターン抽出は初めてだと難しい。デモを丁寧に行い、質問例を提示する |
| セッション3 | 両輪診断で「どちらも低い」と落ち込む参加者がいる可能性。改善の余地があると前向きに捉えさせる |
| セッション4 | 時間が押しやすい。各Partの時間を厳守する |
| セッション5 | 特定メンバーの話になるため、守秘義務を確認。名前は仮名でも可 |
| セッション6 | プランが抽象的にならないよう、「明日から始められるか」を基準に具体化させる |

### 避けるべき表現

| NG表現 | 理由 | OK表現 |
|--------|------|--------|
| 「モチベーションを上げる」 | 抽象的で測定困難 | 「没入パターンが発揮される環境を作る」 |
| 「やる気を引き出す」 | 抽象的 | 「Will/Can/Mustが重なる状態を設計する」 |
| 「楽しく働く」 | 感情と混同する | 「個性が輝く状態で働く」 |
| 「部下を褒める」 | 行動レベルが不明確 | 「輝いていた瞬間を具体的にフィードバックする」 |
