## ワーク詳細

### ワーク1: 没入体験ゲーム「タワービルディング」

**目的**
- 参加者全員が「夢中になる体験」を共通で持つ
- 没入状態を意図的に発生させ、後の振り返りの素材にする

**所要時間**: 30分（説明5分、実施20分、整理5分）

**準備物**
- A4コピー用紙 50枚/グループ
- はさみ 1本/グループ
- セロハンテープ 1個/グループ
- ストップウォッチ（ファシリテーター用）
- 記録用カメラ（スマートフォン可）

**進め方**
1. **ルール説明（5分）**
   - 「紙だけを使って、できるだけ高いタワーを建ててください」
   - 制限時間は15分
   - 紙は折る、切る、丸める、何でもOK
   - 最後に手を離して10秒間自立したら計測
   - 途中で倒れても何度でもやり直しOK

2. **第1ラウンド実施（15分）**
   - タイマーをスタートし、各グループで制作開始
   - ファシリテーターは各グループを観察し、特徴的な行動をメモ
   - 残り5分、残り1分でアナウンス

3. **計測・整理（5分）**
   - 各グループのタワーの高さを計測
   - 順位は発表するが、勝敗にはこだわらない
   - 「今、どんな気持ちでしたか？」と一言ずつ聞く

**ファシリテーターの問いかけ**
- 開始前：「勝ち負けより、"自分が夢中になる瞬間"に意識を向けてください」
- 実施中：（観察に徹し、声かけは最小限に）
- 終了後：「一番集中していたのはどの瞬間でしたか？」

**期待される気づき**
- 「気づいたら時間が過ぎていた」という没入体験
- 自分がどんな役割・行動を取っていたかの自覚
- グループ内での自分の動き方の特徴

**バリエーション**
- 人数が多い場合: 2回戦制にして役割を交代させる
- 時間が短い場合: 制限時間を10分に短縮

### ワーク2: 没入の瞬間を捉える「行動観察シート」

**目的**
- ワーク1での自分の行動を具体的に言語化する
- 没入していた瞬間と、そのときの行動を特定する

**所要時間**: 30分（個人ワーク15分、ペアシェア15分）

**準備物**
- 行動観察シート（A4、1人1枚）
- 筆記用具

**行動観察シートの内容**
```
┌─────────────────────────────────────────────────────────┐
│ 行動観察シート                                         │
├─────────────────────────────────────────────────────────┤
│ 1. 一番夢中になっていた瞬間はいつでしたか？            │
│    □ 最初のアイデア出し  □ 作り始めたとき             │
│    □ 途中で問題が起きたとき  □ 最後の追い込み         │
│    □ その他（                              ）         │
│                                                         │
│ 2. そのとき、あなたは具体的に何をしていましたか？      │
│   （例：「土台の構造を考えていた」「紙を丸めていた」） │
│    →                                                   │
│                                                         │
│ 3. そのとき、どんな感覚がありましたか？                │
│    □ 時間を忘れていた  □ 周りの音が聞こえなかった     │
│    □ 体が勝手に動いていた  □ 次々アイデアが浮かんだ   │
│    □ その他（                              ）         │
│                                                         │
│ 4. グループの中で、あなたはどんな役割でしたか？        │
│    □ リーダー（方向性を決める）                        │
│    □ アイデアマン（発想を出す）                        │
│    □ 職人（手を動かす）                                │
│    □ 調整役（全体を見る）                              │
│    □ 応援団（盛り上げる）                              │
│                                                         │
│ 5. その役割は、自分から取りにいきましたか？            │
│    □ 自然とそうなった  □ 意識して取りにいった         │
│    □ 誰かに任された  □ 他に誰もいなかったから         │
└─────────────────────────────────────────────────────────┘
```

**進め方**
1. **個人ワーク（15分）**
   - 行動観察シートを配布
   - 「さっきのゲーム中の自分を思い出しながら、正直に書いてください」
   - BGMをかけて静かに記入する時間を確保

2. **ペアシェア（15分）**
   - 隣の人とペアになり、お互いのシートを見せ合う
   - 「私は〇〇のときに夢中でした。そのとき△△していました」と共有
   - 相手の話を聞いて「それ、普段の仕事でもやってそう」など気づきを伝え合う

**ファシリテーターの問いかけ**
- 「"正解"はありません。自分の感覚を素直に書いてください」
- 「ペアの人には、"へぇ、そうなんだ"と受け止めてあげてください」
- 「相手の話を聞いて、"それって普段も？"と聞いてみてください」

**期待される気づき**
- 自分が没入する条件（タイミング、状況）の自覚
- 自分が自然と取る役割・行動の自覚
- 他者からのフィードバックによる新たな視点

### ワーク3: 没入パターン抽出「MYフローカード」

**目的**
- 没入する行動パターンを「型」として言語化する
- 自分だけの「没入の取扱説明書」を作成する

**所要時間**: 45分（個人ワーク20分、グループシェア25分）

**準備物**
- MYフローカード（A5サイズ、厚紙）1人1枚
- 没入パターン一覧表（A4、1人1枚）
- カラーペン（グループに1セット）

**没入パターン一覧表（参考リスト）**

| カテゴリ | パターン名 | 特徴 |
|----------|------------|------|
| 創造系 | ゼロイチ型 | 何もないところから作り始めると燃える |
| 創造系 | 改善型 | 既存のものをより良くすることに夢中になる |
| 創造系 | 組み合わせ型 | 異なる要素を組み合わせてアイデアを出す |
| 分析系 | 構造把握型 | 全体像を理解し、仕組みを解明すると集中できる |
| 分析系 | 課題発見型 | 問題点を見つけ出すことに没頭する |
| 分析系 | 最適化型 | 効率を上げる、無駄を省くことに熱中する |
| 実行系 | 職人型 | 手を動かし、形にする作業に没入する |
| 実行系 | 突破型 | 困難な壁を乗り越えることに燃える |
| 実行系 | 完遂型 | 最後までやり切ることに達成感を感じる |
| 対人系 | 調整型 | 人と人をつなぎ、全体をまとめることに集中する |
| 対人系 | 支援型 | 誰かをサポートすることに夢中になる |
| 対人系 | 盛り上げ型 | 場の雰囲気を作ることに没入する |

**進め方**

   - ワーク2の行動観察シートを見ながら、自分のパターンを特定

3. **グループシェア（25分）**

**期待される気づき**
- 自分の没入パターンの言語化
- 没入を「再現」するためのトリガーの特定
- 他者との違い・共通点の発見

### ミニレクチャー: 没入の4要素

**目的**
- 体験から得た気づきを、理論で補強する
- 没入状態を意図的に作るための知識を提供する

**所要時間**: 15分

**準備物**
- スライド（または模造紙）
- 没入の4要素サマリーシート（A4、1人1枚）

**内容**

```
没入（フロー）を生み出す4つの条件

1. 明確なゴール
   - 何を達成すればいいかがはっきりしている
   - 例：「高いタワーを建てる」という明確な目標

2. 即時フィードバック
   - やったことの結果がすぐにわかる
   - 例：タワーが立つ/倒れるがすぐわかる

3. 適切な難易度
   - 簡単すぎず、難しすぎない
   - 自分のスキルよりちょっと上のチャレンジ

4. 自分でコントロールできる感覚
   - やり方を自分で決められる
   - 指示されてやるのではなく、自分で選べる
```

**ファシリテーターの語り例**
「さっきのゲームで没入できたのは、この4つの条件が揃っていたからです。
- ゴールは"高いタワーを建てる"で明確でした
- 立つか倒れるか、フィードバックはすぐわかりました
- 難しすぎず簡単すぎず、ちょうどいいチャレンジでした
- どう作るかは自分たちで決められました

では、あなたの仕事にこの4つの条件は揃っていますか？
揃っていないとしたら、どうすれば揃えられるでしょうか？
この後のワークで、それを考えていきます。」

### ワーク4: 業務棚卸し「没入マッチングシート」

**目的**
- 現在の業務を棚卸しし、没入パターンとの相性を分析する
- 没入しやすい業務・しにくい業務を特定する

**所要時間**: 45分（個人ワーク35分、全体共有10分）

**準備物**
- 没入マッチングシート（A3サイズ、1人1枚）
- MYフローカード（ワーク3で作成したもの）
- 付箋（75mm×75mm、1人20枚程度）
- ペン

**没入マッチングシートの内容**
```
┌────────────────────────────────────────────────────────────────────┐
│                    没入マッチングシート                             │
├────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  ステップ1: 業務の棚卸し（付箋1枚につき1業務）                     │
│  ※週に1回以上やっている業務をすべて書き出す                       │
│                                                                     │
│  ステップ2: マッピング                                             │
│                                                                     │
│         没入しやすい                                                │
│              ↑                                                      │
│              │                                                      │
│    頻度低 ←─┼─→ 頻度高                                           │
│              │                                                      │
│              ↓                                                      │
│         没入しにくい                                                │
│                                                                     │
│  ステップ3: 分析                                                    │
│                                                                     │
│  【右上（没入しやすい×頻度高）】                                   │
│   → 強みを活かせている業務。さらに伸ばす方法は？                  │
│                                                                     │
│  【左上（没入しやすい×頻度低）】                                   │
│   → もっと増やしたい業務。どうすれば増やせる？                    │
│                                                                     │
│  【右下（没入しにくい×頻度高）】                                   │
│   → 工夫が必要な業務。没入パターンを活かす方法は？                │
│                                                                     │
│  【左下（没入しにくい×頻度低）】                                   │
│   → 減らす/手放す候補。本当に必要？                               │
│                                                                     │
└────────────────────────────────────────────────────────────────────┘
```

**進め方**
1. **業務の棚卸し（10分）**
   - 付箋1枚に1業務を書き出す
   - 「週に1回以上やっている業務」を目安に
   - 細かく分ける（「会議」ではなく「○○の会議の司会」など）

2. **マッピング（15分）**
   - シートの4象限に付箋を貼っていく
   - MYフローカードを見ながら「この業務、自分の没入パターンに合っているか？」を判断
   - 迷ったら直感で配置してOK

3. **分析（10分）**
   - 4象限それぞれについて、シートの問いに沿って考える
   - 特に「右下（没入しにくい×頻度高）」に注目
   - 「この業務を、どうすれば没入パターンに合わせられるか？」を考える

4. **全体共有（10分）**
   - 2〜3名に「発見したこと」を共有してもらう
   - 「右下の業務をどう変えようと思うか」を中心に

**ファシリテーターの問いかけ**
- 「業務は"何をするか"だけでなく、"どうやるか"で没入度が変わります」
- 「右下の業務が多くても大丈夫。次のワークで"変え方"を考えます」
- 「左上の業務を増やす交渉を、上司としてみるのも一つの方法です」

**期待される気づき**
- 自分の業務ポートフォリオの可視化
- 没入パターンと業務の相性の自覚
- 「変えられる」という可能性の発見

### ワーク5: 業務リデザイン「没入トリガー設計」

**目的**
- 没入しにくい業務を、没入パターンに合わせて再設計する
- 具体的な「仕掛け」を作り、業務で再現できるようにする

**所要時間**: 45分（個人ワーク20分、ペアコーチング25分）

**準備物**
- 没入トリガー設計シート（A4、1人2枚）
- MYフローカード
- 没入マッチングシート（ワーク4で作成したもの）

**没入トリガー設計シートの内容**
```
┌─────────────────────────────────────────────────────────────┐
│              没入トリガー設計シート                          │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│ ■ 対象業務：                                                │
│   （没入マッチングシートの「右下」から1つ選ぶ）             │
│                                                              │
│ ■ 現状：なぜ没入できていないか？                            │
│   □ ゴールが不明確                                          │
│   □ フィードバックが遅い/ない                               │
│   □ 簡単すぎる/難しすぎる                                   │
│   □ やり方を自分で決められない                              │
│   □ 自分の没入パターンに合っていない                        │
│   □ その他（                          ）                    │
│                                                              │
│ ■ 私の没入パターン：【           】型                       │
│                                                              │
│ ■ リデザイン案                                              │
│                                                              │
│ 【ゴールを変える】                                          │
│   今のゴール：                                               │
│   変更後：                                                   │
│                                                              │
│ 【やり方を変える】                                          │
│   今のやり方：                                               │
│   変更後：                                                   │
│                                                              │
│ 【フィードバックを作る】                                    │
│   今：                                                       │
│   追加する仕掛け：                                           │
│                                                              │
│ 【チャレンジを加える】                                      │
│   今の難易度：                                               │
│   追加するチャレンジ：                                       │
│                                                              │
│ ■ 明日からできる小さな一歩：                                │
│                                                              │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

**進め方**
1. **対象業務の選定（2分）**
   - 没入マッチングシートの「右下（没入しにくい×頻度高）」から1つ選ぶ
   - 一番「変えたい」と思う業務を選ぶ

2. **現状分析（8分）**
   - なぜ没入できていないかを分析
   - 没入の4要素のうち、何が欠けているかを特定

3. **リデザイン案の作成（10分）**
   - 4つの切り口（ゴール/やり方/フィードバック/チャレンジ）から変更案を考える
   - 自分の没入パターンに合わせた変更を意識

4. **ペアコーチング（25分）**
   - 2人ペアになり、1人12分ずつ
   - 発表者：自分のシートを説明（3分）
   - 聞き手：以下の質問でコーチング（9分）
     - 「それ、具体的にはどうやるの？」
     - 「いつからやれそう？」
     - 「うまくいったかどうか、どうやってわかる？」
     - 「それやったら、どんな気持ちになりそう？」

**ファシリテーターの問いかけ**
- 「業務そのものを変えなくても、"取り組み方"を変えるだけで没入度は変わります」
- 「小さな変更でOKです。明日からできることを考えてください」
- 「ペアの人には、"具体的には？"と聞いてあげてください」

**期待される気づき**
- 業務は「変えられる」という感覚
- 没入を「仕掛ける」ことができるという理解
- 具体的な変更アイデアの獲得

**リデザイン例**

| 業務 | 没入パターン | リデザイン案 |
|------|------------|-------------|
| 週次報告書作成 | ゼロイチ型 | 毎回1つ「新しい分析切り口」を入れるルールにする |
| 定例会議の司会 | 調整型 | 参加者の発言量を可視化し、均等になるようファシリする |
| データ入力 | 最適化型 | 前回より5%速くできるかタイムアタックする |
| 顧客対応 | 支援型 | 「この人の一番の困りごとは何か」を毎回1つ特定する |

### ワーク6: アクションプラン「明日からの1歩」

**目的**
- 研修の学びを具体的な行動計画に落とし込む
- 公開宣言により実行確率を高める

**所要時間**: 30分（個人ワーク10分、全体シェア20分）

**準備物**

   - 宣言文を作成

**期待される気づき**
- 学びの具体化・行動化
- 公開宣言による実行へのコミットメント
- 仲間の宣言を聞くことでの刺激

## ファシリテーター向け注意事項

### 全体を通して

1. **「教えない」姿勢を貫く**
   - 正解を与えない。問いかけで気づきを促す
   - 「いい質問ですね」ではなく「どう思いますか？」と返す
   - 参加者同士の対話を促進する

2. **安全な場づくり**
   - 冒頭で「正解はない」「失敗OK」を明言する
   - 否定的なコメントには介入する
   - 全員が発言できるよう配慮する

3. **時間管理**
   - 各ワークの終了5分前にアナウンス
   - 延長は最小限に（全体の流れを優先）
   - 休憩は削らない（集中力維持のため）

### ワーク別の注意点

**ワーク1: タワービルディング**
- 勝敗にこだわりすぎるグループには「没入体験が目的」とリマインド
- 消極的な参加者がいたら「見ているだけでも何か感じたことは？」と声かけ
- 安全管理（はさみの扱い、立ち上がる際の注意）

**ワーク2: 行動観察シート**
- 「書けない」という人には「一番記憶に残っている瞬間は？」と聞く
- ペアシェアで一方的にならないよう、時間を区切る

**ワーク3: MYフローカード**
- パターン一覧表に縛られないよう「自分の言葉でOK」を強調
- 「どれも当てはまらない」という人には「一番近いのは？」と聞く

**ワーク4: 没入マッチングシート**
- 業務の粒度が粗すぎる人には具体例を示す
- 「全部右下」という人には「一番マシなのは？」と聞く

**ワーク5: 没入トリガー設計**
- 「変えられない」という人には「1%だけ変えるなら？」と聞く
- ペアコーチングの質問例を壁に貼っておく

**ワーク6: アクションプラン**
- 宣言を恥ずかしがる人がいても、全員に発表してもらう
- 拍手は全力で。場の雰囲気を盛り上げる

### 想定されるトラブルと対応

| トラブル | 対応 |
|----------|------|
| ゲームに没入できなかった人がいる | 「没入できなかった」ことも発見。「何があれば没入できた？」を考える材料にする |
| グループワークで発言しない人がいる | ペアワークに切り替える、または「書いてから話す」形式にする |
| 業務との接続が難しいという声 | 具体例を複数示す。「完璧でなくていい、仮説でOK」と伝える |
| 時間が押してきた | ワーク5のペアコーチングを短縮（1人8分→6分）、または全体シェアを割愛 |
| 参加者のモチベーションが低い | ワーク1のゲーム体験で巻き込む。座学を最小限にして体験中心で進める |
